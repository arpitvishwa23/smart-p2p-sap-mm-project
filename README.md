# Smart Procure-to-Pay (P2P) Cycle in SAP MM with Vendor Performance Evaluation

## Project Overview
This project is a web-based procurement management system inspired by SAP MM. It manages the complete Procure-to-Pay cycle from requisition to vendor payment and includes vendor performance evaluation.

## Modules
- Login System
- Dashboard
- Purchase Requisition
- Approval Workflow
- RFQ Management
- Purchase Orders
- Goods Receipt
- Invoice Verification
- Vendor Analytics
- Reports
- Master Data

## Technology Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open index.html in browser
3. Or run using Live Server in VS Code

## Unique Feature
Vendor scoring based on delivery, quality, price, responsiveness, and reliability.
